# 告警表达式

## 1. 基础定义

- **表达式**：由多个布尔返回类型的表达式单项通过 `|`、`&` 连接起来的字符串
- **表达式单项**：包含测点、运算规则、延迟时间的字符串，运算结果须为布尔类型（true/false）
- **测点**：应用类型.测点名称（旧模组：设备种类.测点名称）
- **运算规则**：测点值需要满足的运算规则（详见下方函数库）
- **延迟时间**：测点值的延迟时间，满足业务场景对周期运算规则的需要

## 2. 函数库

本节总体给出系统目前支持的函数及使用规则，需注意，**函数区分大小写**。

假设：

- 设备类型/应用类型为：`dt1`、`dt2`
- 测点名称为：`pn1`、`pn2`
- 延迟(周期)时间为：`0`（不延迟）、`d`（延迟d秒）

## 3. 函数具体示例

本节总体给出系统目前支持函数的具体示例，方便理解。

### 3.1 `==` 等于

- **模组**：模组-A-01
- **告警名称**：动环数据采集器通讯中断
- **设备举例**：`DEV-A-001`
- **表达式**：`[{动环数据采集器.通讯状态}==1:0]`
- **描述**：当DEV-A-001在当前时刻的测点"通讯状态"值为1时，触发告警

### 3.2 `>` 大于

- **模组**：模组-A-01
- **告警名称**：弱电电池组放电电流不均
- **设备举例**：`DEV-A-002`
- **表达式**：`[{弱电电池架.电流不均衡度}>0.1:30]`
- **描述**：当DEV-A-002在当前时刻和30秒前的测点"电流不均衡度"值大于0.1时，触发告警（仅需30秒前的值和当前值满足大于0.1的条件，即可触发，中间无需一直满足）

### 3.3 `>=` 大于等于

- **模组**：模组-A-01
- **告警名称**：柴发并机输出柜接地告警
- **设备举例**：`DEV-A-003`
- **表达式**：`[{柴发并机输出柜.综保零序电流}>=10:0]`
- **描述**：当DEV-A-003在当前时刻的测点"综保零序电流"值大于或等于10时，触发告警

### 3.4 `<` 小于

- **模组**：模组-B-01
- **告警名称**：1#纯水箱液位低告警
- **设备举例**：`DEV-B-001`
- **表达式**：`[{空调RO水处理设备.纯水箱液位百分比_1}<30:0]`
- **描述**：当DEV-B-001在当前时刻的测点"纯水箱液位百分比_1"值小于30时，触发告警

### 3.5 `<=` 小于等于

- **模组**：模组-B-01
- **告警名称**：中压方仓直流屏电池柜电池组放电
- **设备举例**：`DEV-B-002`
- **表达式**：`[{中压直流屏电池柜.电池组电流}<=-3:0]`
- **描述**：当DEV-B-002在当前时刻的测点"电池组电流"值小于或等于-3时，触发告警

### 3.6 `!=` 不等于

- **模组**：模组-B-02
- **告警名称**：柴油发电机组启机柴发百叶未打开
- **设备举例**：`DEV-B-003`
- **表达式**：`[{柴油发电机组.柴发百叶开关状态}==0:10]&[{柴油发电机组.发电机运行状态}!=0:10]`
- **描述**：当DEV-B-003在当前时刻和10秒前的测点："柴发百叶开关状态"值等于0时，并且测点"发电机运行状态"不等于0时，触发告警（仅需10秒前的值和当前值满足条件，即可触发，中间无需一直满足）

    简单理解，在当前和10秒前的时刻，柴发百叶开关状态等于0时，并且发电机运行状态不等于0时，即可触发

### 3.7 `===` 恒等于

- **模组**：模组-A-01
- **告警名称**：控制器电源异常告警
- **设备举例**：`DEV-A-004`
- **表达式**：`[{输入断路器.控制器电源异常告警}===1:60]`
- **描述**：当DEV-A-004在当前时刻到60秒前的测点"控制器电源异常告警"值恒等于1时，触发告警（注意，此处是恒等于，当前到60秒前的测点值需要一直保持为1的状态，才可触发告警）

### 3.8 `!==` 恒不等于

- **模组**：暂未使用

### 3.10 `>>=` 恒大于等于

- **告警名称**：中压方仓温度过高

- **设备举例**：`DEV-D-001`

### 3.22 sum 计算总和

- **模组**：模组-E-02
- **告警名称**：冷冻供水总管单路流量中断
- **设备举例**：单系统下的末端主管流量计 `DEV-E-003`、`DEV-E-004`、`DEV-E-005`、`DEV-E-006`，关联的BA群控制器柜 `DEV-E-007`
- **表达式**（恢复表达式）：`sum([{单系统末端主管流量计.实时流量}:0])>0.9*([{BA群控制器柜.末端实时负载}:0]/1.163/([{BA群控制器柜.冷冻主管供回水温差_1}:0]+[{BA群控制器柜.冷冻主管供回水温差_2}:0])/2)`
- **描述**：当单系统下的四个末端主管流量计的测点"实时流量"在当前时刻的总和，大于 `0.9 × BA群控制器柜"末端实时负载" / 1.163 / ("冷冻主管供回水温差_1"与"冷冻主管供回水温差_2"的平均值)` 时，表达式成立

    简单解释，如果表达式成立，说明实际流量充足，告警可恢复；若不成立，则可能表示流量不足，需要检查水泵、阀门或过滤器状态。
