## TBOS

TBOS (T-Block Operation System) 是腾讯自研的机房动环监控系统，覆盖设备接入、数据采集、实时告警、配置管理与可视化等完整链路。

本文档面向**对接方工程师**，帮助您快速完成系统部署、设备接入、告警配置及第三方系统对接。

### 1. 亮点

- 🔌 **多协议采集** — Modbus RTU/TCP、SNMP、DIO，可扩展驱动插件
- ⚡ **实时告警** — 毫秒级响应，TNQL 表达式引擎支持复杂告警逻辑
- 🔄 **分布式架构** — 水平扩展、高可用，主备 Kafka 双活自动切换
- 🌐 **边云协同** — 边缘采集 + 云端管控，支持配置本地缓存兜底
