# TBOX 采集对接

## 需求背景

腾讯提供 TBOX（物联网采集网关硬件）作为标准采集终端。厂商需按照腾讯提供的 TBOX 通信协议与数据规范，实现 TBOS 平台与 TBOX 设备的对接，完成传感器数据采集上送。这是合作厂商与腾讯在技术层面的唯一强制交集。

## 阻塞原因

- 目前缺少真实环境的 TBOX 硬件与传感器用于联调验证
- TBOX 通信协议文档尚未对合作厂商正式发布
- 采集数据模型（测点模板、点位表）尚未定义

## 待明确事项

- [ ] 支持的传感器类型与测点模板清单
- [ ] TBOX 设备注册与鉴权流程
- [ ] 采集数据上送的频率、批量策略与断点续传机制
