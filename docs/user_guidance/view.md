# 视图功能开发

## 需求背景

合作厂商需基于 TBOS 框架自行开发可视化视图功能，包括设备状态面板、实时数据展示、告警大屏等前端页面。TBOS 开源版不提供现成视图，厂商需根据自身运维场景定制。

## 阻塞原因

- 运营商侧未给出具体的视图功能清单与 UI 原型
- TBOS 开源版尚未定义前端框架约束与组件标准
- 不同厂商的运维场景差异大，缺乏统一的定制范式
